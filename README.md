# Studio Barnuevo — Renovation Architecture

A modern, minimal portfolio website for Studio Barnuevo, a renovation architecture studio.

## Features

- Full-screen hero with subtle parallax
- Project grid with category filtering
- Scroll-reveal animations
- Animated statistics counters
- Contact form
- Fully responsive (mobile, tablet, desktop)
- Zero dependencies — pure HTML, CSS, and vanilla JavaScript

## Getting Started

Simply open `index.html` in any browser, or deploy to any static hosting service (GitHub Pages, Netlify, Vercel).

## Deploying to GitHub Pages

1. Go to your repo **Settings → Pages**
2. Set source to **Deploy from a branch**
3. Select **main** branch, root folder
4. Your site will be live at `https://yourusername.github.io/mediakit-website/`

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Grid, Flexbox, Animations)
- Vanilla JavaScript (IntersectionObserver, smooth scroll)
- Google Fonts (Inter + Playfair Display)
- Unsplash images (replace with your own project photos)
